Before try compiling LuaJ,
you need install its dependencies:

- Install Java JDK
- GNU Make
- GCC

Compile LuaJ with:   make


-------------------------------------------
If you wish, install it globally.


1. Check some directories with: echo $PATH

2. Copy "luaj-jse-3.0.2.jar" and "luaj"
to one of all listed directories.



Example on Ubuntu 22.x: 
*Default pattern: dir_path1:dir_path2:dir_path3: ...

/usr/local/sbin
/usr/local/bin
/usr/sbin
/usr/bin
/sbin
/bin
/usr/games
/usr/local/games
/snap/bin
/snap/bin
